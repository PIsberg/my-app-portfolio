# Purpose
Portfolio Android App that includes projects from Nanodegree in Android programming @ Udacity

# Build Instructions
This project uses Gradle. To build the project, run the following command in the root directory:

## Windows
```powershell
.\gradlew.bat clean assembleDebug
```

## Linux / Mac
```bash
./gradlew clean assembleDebug
```

The APK will be generated in `app/build/outputs/apk/debug/app-debug.apk`.
